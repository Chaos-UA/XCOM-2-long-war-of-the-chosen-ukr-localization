Better UI Fonts

EN:

This mod improves default UI fonts (Cyrillic, mostly). Changes:

1. "Alpha" font – all characters now have more consistent look. Cyrillic characters and some extended Latin characters initially were taken from other font that even doesn't have height matched.
2. "AF Carplates Medium" font – it already looks fine aside from missing gaps between Cyrillc characters. The mod adds them, so font now looks same in both Latin and Cyrillic.

Due the nature of how font mods work, this mod is incompatible with other UI font mods.

RU:

Этот мод улучшает стандартные шрифты интерфейса (в основном кириллицу). Изменения:

1. Шрифт "Alpha" – все символы теперь выглядят похоже. Кириллица и часть расширенной латиницы была взята из другого шрифта, у которого даже высота не совпадает.
2. Шрифт "AF Carplates Medium" – он и так выглядит нормально, если не считать пропущенные отступы у кириллицы. В моде отступы добавлены, поэтому шрифт выглядит одинаково и на латинице, и на кириллице.

Из-за того, как работают моды на шрифты, этот мод несовместим с другими модами на шрифты интерфейса.
